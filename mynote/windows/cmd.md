## 1、常用cmd命令：

```c
dir                                                      //显示所有文件与目录
D:                                                       //进入D盘
cd <dir>                                                 //进入目录
cls                                                      //清除目录
del <file>                                               //删除文件
```

## 2、cmd自定义字体：

- 推荐字体：Microsoft YaHei Mono，下载后安装即可，之后在cmd中设置字体为Microsoft YaHei Mono；

- cmd：右键 -> 属性 -> 字体 -> Microsoft YaHei Mono；
