# 1、Mobaxterm常用快捷键：
```bash
Ctrl + Shift + B                                                     # 显示/隐藏侧边栏
Ctrl + L                                                             # 清除画面
Ctrl + Tab                                                           # 向前导航窗口
Ctrl + Shift + Tab                                                   # 向后导航

# 自定义快捷键切换标签
Setting -> Keyboard shortcuts -> Previous tab -> Ctrl + Shift + LEFT
                              -> Next tab     -> Ctrl + Shift + RIGHT
```

# 2、常用设置：

```bash
# 设置右键为快捷菜单
Setting -> Configuration -> Terminal -> Terminal features -> (取消勾选)Paste using right-click

# Mobaxterm设置home目录和根目录
Setting -> Configuration -> General -> Persistent home directory     -> _MobaFolder_\home
                                    -> Persistent root (/) directory -> _MobaFolder_\slash

# 只显示菜单栏
View -> Show menu bar

# ssh建立连接后不打开侧边栏
Setting -> Configuration -> SSH -> (取消勾选)Automatically switch to SSH-browser tab after login

# 允许打开多个Mobaxterm窗口
Setting -> Configuration -> Misc -> Allow multiple instances of MobaXterm -> OK
```
