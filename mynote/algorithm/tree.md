# 二叉树

## 1、二叉搜索树：

### 1）、定义：

* 定义：左子树每个节点的值都不大于父节点，右子树所有节点的值都不小于父节点。

* 别名：二叉查找树

* 特性：二叉树中序遍历节点为升序

* 最大值：最左子节点

* 最小值：最右子节点

```mermaid
graph TB;
1((15))
2((10))
3((20))
4((4))
5((11))
6((18))
7((25))
1-->2
1-->3
2-->4
2-->5
3-->6
3-->7
```
