# 1、style属性：

```bash
position  # 定位
	absolute  # 绝对定位，相对于具有position属性的父元素的定位
	relative  # 相对定位，相对自己原本的位置的定位

margin  # 外边距
	margin: auto;  # 自动，依赖于浏览器
	margin-top: 100px;
	margin-bottom: 100px;
	margin-right: 50px;
	margin-left: 50px;
	margin: 25px 50px 75px 100px;  # 上、下、左、右边距
```

# 2、div属性：

```bash
style
	position
	left

align  # 对齐
	left  # 左
	right  # 右
	center  # 居中
	justify  # 行伸展
```
