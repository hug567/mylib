# 1、常用命令：

```bash
tree                      # 树状显示当前目录及子目录中所有目录与文件
  -L <num>                  # 指定显示深度
  -d                        # 只显示目录
  -f                        # 显示完整相对路径
  -i                        # 不以阶梯状显示
  -n                        # 不增加色彩显示
  -p                        # 显示权限
  -s                        # 显示文件或目录大小
  -t                        # 已更改时间排序
  -u                        # 显示所属用户
  -g                        # 显示所属用户组
```
