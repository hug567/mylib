# 1、常用命令：

```bash
# 安装
sudo apt install p7zip-full

7z
    x        # 解压文件
    -o"dir"  # 解压到指定目录
    a        # 压缩文件
    l        # 查看压缩包文件
    t        # 测试压缩文件
```

# 2、常见用法：

```bash
# 解压7z文件到当前目录
7z x file.7z
# 解压到指定目录，-o后不能有空格
7z x file.7z -o"dir"

# 压缩文件
7z a file.7z file1 file2 dir1 ...

# 测试压缩文件，查看相关信息
7z t file.7z

# 查看压缩包内容
7z l file.7z
```
