# 1、安装：

```bash
# ubuntu
sudo apt install sqlite3
```

# 2、常用命令：

```bash
sqlite3
	--help                                  # 查看帮助
	--version                               # 查看版本
```

# 3、常见用法：

```bash
# 进入数据库
sqlite3 <file>
	.help                                   # 查看帮助
	.database                               # 查看数据库信息
	.table                                  # 查看所有表
	.quit                                   # 退出
```
