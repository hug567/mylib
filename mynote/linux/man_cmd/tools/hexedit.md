# 1、命令参数：

```bash
# 安装
sudo apt install hexedit

hexedit
	-l <num>                  # 打开文件指定列宽
```

# 2、常用命令：

```bash
man hexedit                                              # 查看手册

hexedit <filename>                                       # 打开文件
hexedit -l 16 <filename>                                 # 打开文件时指定列宽
F1                                                       # 帮助
F2                                                       # 保存
F7                                                       # 复制
F8                                                       #
Ctrl + X                                                 # 保存并退出
Ctrl + C                                                 # 退出不保存
PageUp/PageDown                                          # 翻页
</>                                                      # 跳转到文件：开头/结尾
```
