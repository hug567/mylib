# 测试网络端口是否开放

## 1、常用参数：

```bash
nmat
	-h                        # 查看帮助
	-p <port>                 # 指定端口
	-Pn                       # 认为对端机器均在线，不进行在线检测
	-sU                       # 测试udp端口是否开放
```

## 2、常见用法：

```bash
# 测试指定ip的所有端口是否开放
nmap 192.168.0.3
# 跳过在线检测，测试指定ip的所有端口是否开放
nmap -Pn 192.168.0.3
# 跳过在线检测，测试指定ip的指定端口是否开放
nmap -Pn 192.168.0.3 -p 21115
# 跳过在线检测，测试指定ip的指定范围内的端口是否开放
nmap -Pn 192.168.0.3 -p 21115-21119
# 跳过在线检测，测试指定ip的指定范围内的端口UDP是否开放
nmap -Pn -sU 192.168.0.3 -p 21115-21119
```
