# 1、常用参数：

```bash
du
	--help                            # 查看帮助
	-h                                # 以易读方式打印大小
	-s                                # 只显示总大小
```

# 2、常见用法：

```bash
# 查看当前目录总大小
du -sh
# 查看指定文件大小
du -sh <file>
# 查看指定目录大小
du -sh <dir>
# 查看当前目录所有文件和目录的大小
du -sh *
# 查看当前目录所有隐藏文件和目录的大小
du -sh .*
# 查看当前目录总大小时忽略指定目录
du -sh --exclude=.git
```
