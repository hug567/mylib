# 1、常用命令：

```bash
sort
	--help                                # 查看帮助
	-b                                    # 忽略每行开始前的空格
	-f                                    # 排序时将小写字母视为大写字母
	-n                                    # 依照数值大小排序
	-u                                    # 去重
	-r                                    # 以相反顺序排序
```

# 2、常见用法：

```bash
# 默认按字符串升序排序
sort
# 排序的同时去重
sort -u
```
