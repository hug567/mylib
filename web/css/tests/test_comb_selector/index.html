1、后代选择器：空格

后代选择器：p是div的直接后代，应用了后代选择器样式

后代选择器：p是div的多层后代，也应用了后代选择器样式

对比：p不是div的后代

2、子元素选择器：>

子元素选择器：p是div的直接子元素，应用了子元素选择器样式

对比：p是div的多层后代，但不是直接子元素，没有应用子元素选择器样式

3、相邻兄弟选择器：+

div内部元素，未应用相邻兄弟选择器样式

相邻兄弟选择器：p是div的第一个相邻兄弟，应用了相邻兄弟选择器样式

对比：p是div第二个相邻兄弟，不是第一个，没有应用相邻兄弟选择器样式

4、后续兄弟选择器：~

div内部元素，未应用后续兄弟选择器样式

后续兄弟选择器：p是div的第一个后续兄弟，应用了后续兄弟选择器样式

后续兄弟选择器：p是div的第二个后续兄弟，也应用了后续兄弟选择器样式

对比：p是div后续兄弟的子元素，不是后续兄弟，没有应用后续兄弟选择器样式

5、组合多样式选择器：

A：未应用样式
B：未应用样式

C：A的子元素B，B的后续兄弟元素C，C应用指定样式 (follow01-div follow02-div ~ follow01-p)

6、两个不同选择器有相同属性：

sub01-div > sub01-p, 背景色是：#ff90bc

adjacent01-div > adjacent01-p, 背景色是：#ff90bc
